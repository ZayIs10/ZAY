```
Take Video 1 (the attached screen recording) as the main source footage. Trim and

- Visual: 10-second segment cycling through 3 different account-profile or

- Text overlay — 3-line reveal, each line lands ~3.3s apart, stays through end:
  Line 1 (NEON GREEN): "1. SHARES"
  Line 2 (white): "2. WATCH TIME"
  Line 3 (NEON GREEN): "3. SAVES"
- Voiceover: "Shares first. Watch time second. Saves third. None of these accounts

- End frame: hold last frame for 0.5 seconds for cover-frame selection.
```
