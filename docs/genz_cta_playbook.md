# GenZ Capital — CTA Playbook (Beat 6 + caption)

The exact lines to end every reel with, why, and when to use each. Implements
the strategy in [genz_brand_positioning.md](genz_brand_positioning.md) §7.

Two surfaces carry a CTA:
1. **Beat 6 on-screen** (the `verb / payload / handle` stack from the reel format
   spec, [genz_reel_format.md](genz_reel_format.md) §2). ~3 seconds. Read muted.
2. **The caption** (Post Caption field). Where the *follow* ask + keyword + 1 line.

These are different jobs — see §5. Don't paste the caption CTA onto the screen.

---

## 0. The two rules that override everything

1. **On Shorts, the CTA is a behavior you engineer, not a sentence you append.**
   The four behaviors that grow the channel, by leverage:
   **Follow > Rewatch(loop) > Comment > Save/Send.** Pick ONE per reel and build
   the ending around causing it. Asking for all four = causing none.

2. **Never use bare "follow for more" or "like and subscribe."** IG suppresses
   the first as bait; both are weak everywhere. Always name the *specific value*
   of following (see [genz_brand_positioning.md] §6). Specific = safe AND better.

## 1. Phase 1 CTA menu (USE NOW — follow / loop only)

Pick by what the reel did. All are safe on YouTube **and** Instagram.

### A. The "I do the work" follow (DEFAULT — use when unsure)
On-screen (Beat 6, 3-line stack):
```
FOLLOW
I FIND THESE
SO YOU DON'T HAVE TO
```
Spoken VO line: *"Follow GenZ Capital — I find these so you don't have to."*
Why: states the exact reason to follow (you do the research). This is the brand
promise verbatim. Highest-trust follow ask we have.

### B. The series follow (use when the reel is part of a numbered run)
On-screen:
```
PART [N]
THE REST ARE
ON THE PAGE
```
VO: *"This was part [N]. The rest are on the page."*
Why: follows are driven by fear-of-missing-the-next-one. Best long-term follow
driver. Pair with a series naming convention (e.g. "AI tricks #__").

### C. The loop close (use when the reel teaches ONE prompt/trick)
On-screen: re-state the hook's promise as a payoff that sends them back to 0:00.
Example — if the hook was "Your ChatGPT answers are mid. Here's the fix" →
```
NOW GO
PASTE IT
↑ PROMPT'S AT THE START
```
VO: *"Paste it. Watch the start again if you missed it."*
Why: rewatches are the single strongest reach signal (replays count double on IG,
boost avg-view-% on YT). No follow ask — the loop IS the move.

### D. The comment close (use when the topic is debatable / opinion-y)
On-screen:
```
WHICH ONE
WOULD YOU
ACTUALLY USE?
```
VO: *"Which one would you actually use? Tell me."*
Why: comments >3 words are a strong signal on both platforms, and a real question
(not "comment YES") is NOT bait. Drives reach without a follow ask.

### E. The save close (use when the reel hands over a list / prompt / steps)
On-screen:
```
SAVE THIS
BEFORE YOU
LOSE THE PROMPT
```
VO: *"Save this before you lose the prompt."*
Why: AI-tips content is inherently save-worthy. Saves-per-reach is a top-3 IG
signal and a strong YT signal. Genuine utility, never reads as bait.

## 2. Phase 2 CTA menu (DO NOT USE until a destination exists)

Locked behind the §7 trigger in the positioning doc. When flipped, use on
**~1 in 4 reels max** — never every reel, or it trains the audience to ignore it.

### F. Long-video funnel (trigger: ≥3 long YouTube videos exist)
On-screen:
```
FULL BREAKDOWN
15-MIN VERSION
→ ON MY CHANNEL
```
VO: *"Want the full breakdown? Fifteen-minute version's on my channel."*

### G. Freebie funnel (trigger: a hosted freebie exists)
On-screen:
```
THE PROMPT PACK
IS FREE
→ LINK IN BIO
```
VO: *"All [N] prompts are free — link in my bio."*
Note: on Shorts, push them to the **profile/bio**, not "description" — Shorts
viewers tap profiles far more than descriptions.

## 3. How to PICK the CTA for a given reel (decision order)

Run top-to-bottom; use the first that matches:

1. Is it part of a numbered series? → **B (series follow)**
2. Phase 2 live AND this is a ~1-in-4 "funnel" reel? → **F or G**
3. Does it teach ONE copy-paste prompt/trick? → **C (loop)**
4. Is the topic an opinion / "which is better"? → **D (comment)**
5. Does it hand over a list / multi-step thing? → **E (save)**
6. None of the above / unsure → **A (the default follow)**

One CTA per reel. Never stack "follow AND save AND comment."

## 4. Writing rules for the on-screen Beat-6 stack

- **3 lines, ≤3 words per line.** It's on screen ~3s and read muted. Long = unread.
- **Anton uppercase**, same as every other punch (per reel format spec).
- Line 2 (the payload) can be **neon green** to match the punch language.
- Must fit 1080px wide at the punch font size — same constraint as all reel text
  (see [feedback_reel_text](../../.claude/projects/c--Users-Marc-Desktop-Gen-Z-autamation/memory/feedback_reel_text.md)).
- The VO line must mirror the on-screen text closely (the word-for-word rule,
  same memory). Don't show one thing and say another.

## 5. The caption CTA (different job from on-screen)

The caption is for **SEO + the follow ask**, not the loop/save mechanics.
Template:

```
[Topic keyword in the FIRST sentence — this is the search signal.]

[1 sentence of payoff/why-it-matters.]

Follow @genzcapital — [specific value, e.g. "the AI tricks worth your time, daily"].

#3to5 #relevant #hashtags #max
```

Rules (from [instagram_growth_framework.md] §0.7):
- Topic keyword in sentence 1 (SEO outranks hashtags now).
- 3–5 hashtags MAX (more = spam flag).
- The follow ask here may be explicit because captions aren't scanned for bait
  the way on-screen text is — but STILL name the specific value, never bare
  "follow for more."

## 6. What NOT to do (learned / rule-derived)

- ❌ "Link in description" before Phase 2 — leaks attention, converts ~nobody at
  low follower counts, costs watch-time signal. (positioning §7)
- ❌ "Follow for more" / "like and subscribe" — IG bait flag + weak. (§0)
- ❌ More than one ask per reel — splits the action, causes none.
- ❌ A link CTA on more than ~25% of reels even in Phase 2 — audience tunes it out.
- ❌ On-screen CTA longer than 3 lines / 3 words per line — unread.
- ❌ Changing the brand promise per reel — the follow reason ("I find these so
  you don't have to") is constant; only the *mechanic* (follow/loop/comment/save)
  rotates.
